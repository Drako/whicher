abstract fun resolve(dir: Path, file: Path): List<Path> (source)

Concatenate dir and file and resolve them to an absolute path. This does not check for the existence of the file.

Parameters

dir - The directory in which the file is located. file - The name of the file.

Return
The absolute path to the file including the filename as a list. (Can contain multiple files depending on the platform.)
